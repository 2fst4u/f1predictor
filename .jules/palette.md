## 2026-01-07 - Accessibility in Data Tables
**Learning:** Using `<abbr>` tags with `title` attributes in table headers is a lightweight way to explain technical abbreviations without cluttering the UI. Adding `scope="col"` to `<th>` elements is critical for screen reader users to understand the relationship between headers and data cells.
**Action:** When working with data-heavy tables, always ensure abbreviated headers have semantic explanations and proper scope attributes.
